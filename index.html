Step 1: Check Firebase Configuration

Step 2: Check Firestore Security Rules

The most common issue is security rules. You need to allow read/write access.

Go to Firebase Console → Firestore Database → Rules tab

Paste these rules and click "Publish":

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Note: These rules allow anyone to read and write to your database. For production, you should implement proper authentication.
