Troubleshooting Steps

                
                

                    

                        1 Check Firestore Security Rules
                    

                    
The most common issue is with Firestore security rules. You need to allow read/write access.

                    

                        
Go to Firebase Console → Firestore Database → Rules tab and paste these rules:

                        

                            
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

                            
                        

                    

                    

                        Note: These rules allow anyone to read and write to your database. For production, implement proper authentication.
                    

                

                
                

                    

                        2 Enable Firestore Database
                    

                    
Make sure you've created and enabled Firestore in your Firebase project:

                    
    
                        
  1. Go to the Firebase Console
    2. 
                        
  2. Select your project
    3. 
                        
  3. Click on "Firestore Database" in the left sidebar
    4. 
                        
  4. Click "Create Database" if you haven't already
    5. 
                        
  5. Choose a location and start in test mode
